## Donation Centers Map – Skopje

This project represents a web application that displays donation centers in Skopje using an interactive map.

The goal of the application is to help users easily find organizations that accept donations such as food, clothes and other basic supplies.

The map displays different humanitarian organizations and institutions where citizens can donate and help people in need. By clicking on a location marker, users can see a short description of the organization and the type of donations it accepts.

## Technologies Used

- HTML
- CSS
- JavaScript
- JSON
- Leaflet.js

## Data

The application contains data about several donation centers in Skopje including their name, category, description and geographic coordinates.
